# 🎉 김지민 복귀 축하 사이트

김지민님의 복귀를 축하하기 위한 인터랙티브 웹 페이지입니다.

## ✨ 주요 기능

- 🎆 **폭죽 애니메이션** — "축포 터뜨리기" 버튼을 누르거나 화면 아무 곳이나 클릭하면 폭죽이 터집니다.
- ⭐ **반짝이는 별 배경** — 밤하늘처럼 반짝이는 별들이 배경을 수놓습니다.
- 🎈 **떠오르는 축하 이모지** — 화면 아래에서 축하 이모지가 계속 떠오릅니다.
- 💙 **글래스모피즘 축하 카드** — 부드럽게 등장하는 환영 메시지 카드.
- 📱 **반응형 디자인** — 모바일·데스크톱 모두에서 예쁘게 보입니다.

## 📁 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 페이지 구조 |
| `style.css` | 스타일 및 애니메이션 |
| `script.js` | 폭죽·별·이모지 인터랙션 |

## 🚀 실행 방법

### 로컬에서 보기
`index.html` 파일을 브라우저로 열면 됩니다.

### GitHub Pages
이 저장소는 GitHub Actions를 통해 GitHub Pages로 자동 배포됩니다.
`.github/workflows/deploy-pages.yml` 워크플로가 정적 파일을 배포합니다.

배포 후 주소: `https://sangkwon9.github.io/guyomibirthday/`
